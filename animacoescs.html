O que são animações CSS ?


Você pode alterar quanta propriedade CSS quiser quantas vezes quiser.

para usar animação CSS, você ddeve primeiro especificar alguns quadros -chave para a animação.

Os quadros-chave contêm quais estilos o elemento terá em determinado momento.s


A regra @Keyframes


Quando você especifica estilos CSS dentro da @Keyframes regra, a animação mudará gradualmente do estilo atual para o novo estilo em determinados momentos.

para que uma animação funcione,você deve vinculá-la a um elemento.


Execute a anamação na direção reversa ou em ciclos alternativos

A animation-direction propriedade específica se uma animação deve ser reproduzida para frente, para trás ou em ciclos alternados.


A propriedade animation-direction pode ter os seguintes valores:


  • normal- A animação é reproduzida normalmente(avançar). Este é o padrão
  • reverse-A animação é reproduzida na direção reversa (para trás)
  • alternate- A animação é reproduzida primeiro para a frente e depois para trás
  • alternate-reverse- A animação é reproduzida primeiro para trás e depois para frente
